# 크론탭의 개념

## 1.1 크론탭이란?
```
크론탭(Crontab)은 리눅스 운영체제에서 배치 작업을 스케쥴링 하기 위한 프로그램
크론탭을 이용하면 특정 시간에 작업이 실행되게끔 할 수도 있고, 정기적인 작업 스케쥴링을 할 수 있다.
```
> I Think .. 크론탭을 이용해서 자사의 php 스크립트를 실행시켜 메일로 원하는 시간마다 원하는 정보를 받아 볼 수 있었다.
